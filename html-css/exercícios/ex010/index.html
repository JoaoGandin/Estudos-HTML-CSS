<a href="https://github.com/JoaoGandin" target="_blank" rel="external">

<a> é a tag de âncora usada para criar links, direcionando o user para outras páginas internas ou externas. Esse href="" ai é aonde vai colocar o link de referencia. Target="_blank" é para abrir o link externo em uma nova guia. Esse rel="external" é para indicar que o link é um link externo.

aqui você pode usar o rel="next" que indica que o link leva para a próxima página e se caso você quiser voltar para a página anterior você indica com rel="prev" (prev = previous)

rel=""

esse rel ai é necessario, pois ele indica para o google qual é a próxima pág. ou a pág anterior ou externa, ele da um valor para o link, ele tem uma sêmantica

  • rel="external"

    • indica que o link é um link externo

  • rel="next"

    • indica que o link leva para um outro arquivo da página

  • rel="prev"

    • indica que o link volta para o outro arquivo da página

  • rel="nofollow"

    • indica que o mecanismo de busca não precisa indexar esse site. Você está dizendo para o mecanismo de busca que você não da um aval para esse conteúdo.
