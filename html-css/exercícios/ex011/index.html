Imagem dinâmica

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador

A tag <picture> é utilizada para permitir a exibição de imagens responsivas

<source>: O navegador avalia cada elemento <source> dentro de <picture> em ordem. Ele seleciona o primeiro que corresponde às condições especificadas (geralmente através de atributos media ou type) e carrega o arquivo de imagem correspondente. Se nenhuma das tags <source> for compatível, ele usa o src da tag <img> aninhada como fallback.
