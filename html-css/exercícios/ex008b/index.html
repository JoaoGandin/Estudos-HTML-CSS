Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript. A tag usada foi o <code>. Mas a tag <code> não faz a identação, por exemplo:

num = int(input('Difite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é IMPAR') print('Fim do programa')

para fazer uma identação utiliza-se a tag <pre>, tudo que estiver dentro do <pre> ele irá considerar, quebra de linhas, tabulação, quebra de linhas, etc. Por exemplo: 

      
  num = int(input('Difite um número')) 

  if num % 2 == 0: 
    print(f'O número{num} é PAR') 
  else: 
    print(f'O número {num} é IMPAR') 
  print('Fim do programa')
