Listas Ordenadas <ol>

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho

O <ol> significa que é uma lista ordenada e para colocar os itens dentro dessa lista você utiliza o <li>

as listas tem uns parâmetros que pode colocar no <ol> que é o type="" onde tem vários tipos de marcação, como:

  • type="1" (padrão) (números)
  • type="A" (letras maiúsculas)
  • type="a" (letras minúsculas)
  • type="I" (números romanos maiúsculos)
  • type="i" (números romanos minúsculos)
O start="" serve para definir o número inicial da lista. Porém quando você for usar por exemplo o type="a".
O start="" tambem tem que ser númerico por exemplo: <ol type="a" start="3"> (vai começar do "c" que é o terceiro item do alfabeto)

Descrição

O <dl> significa que é uma description list e para colocar os itens/termos dentro dessa lista você utiliza o <dt>. que define o termo (nome) e o <dd>. que define a descrição do termo.

Por exemplo:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
